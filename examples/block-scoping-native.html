Block-level scoping with let and const (Native)

This page runs a series of a examples using block-level scoping with let and const in ECMAScript 6 without any transpilation. This means your browser must support ES6 let and const in order to successfully run the code. Open up your debugging console to view the output.

Browser support: Chrome, Firefox & Edge

For more information on how arrow functions work and their level of support, check out the Block-level scoping with let and const article that is part of the Learning ES6 series.
