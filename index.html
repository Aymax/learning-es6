Learning ES6

Below you will find code examples for the different features of ES6 based on the Learning ES6 blog series. All of the examples use browser-based transpilation of ES6 code to ES5 for demonstration purposes. If you want to use ES6 in a production environment, you should use statical transpilation. See Using ES6 right now for more information.

Check out the learning-es6 Github repository for the source code.

Examples:

  • Arrow functions - a short-hand form of an anonymous function
    • Native example
    • Babel example
    • Traceur example
  • Block scoping - scoping variables to blocks via let keyword
  • Destructuring - A concise yet flexible way of mapping values from objects or arrays into variables
  • Parameter handling - more ways to deal with the arguments to a function
  • Object literal features - various shorthand syntaxes for building up object literals
  • Template literals - much cleaner way to build up string values
  • Promises - an alternative to callback functions for handling asynchronous operations
  • Generators - specialized functions that create iterators using yield keyword
  • Classes - a formalized approach to define classes in JavaScript
  • Modules - provide a modular of organizing and loading JavaScript code
  • Maps - dictionary-type object that can store key/value pairs
  • Sets - A collection object that can store a unique list of values
  • Iterables and Iterators - a new ES6 interface for iteration
  • New APIs - new APIs for existing native JavaScript classes Math, Object, RegExp, etc.
